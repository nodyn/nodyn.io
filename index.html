Nodyn

Nodyn is a node.js compatible framework, running on the JVM powered by the DynJS Javascript runtime running under vert.x - the polyglot application platform and event bus.

Imagine running your Node.js app on the JVM with clean, clear access to Java directly in your Javascript. All that with vert.x's powerful clustering technology built right in.

Open Source

Nodyn is an open source project, distributed with the Apache license. You can find the Nodyn source code on GitHub.

The Nodyn project is produced by the Project:odd team, and a part of the JBoss polyglot ecosystem.
