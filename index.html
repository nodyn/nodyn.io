Nodyn

Nodyn is a Node.js compatible framework, running on the JVM powered by the DynJS Javascript runtime.

Open Source

Nodyn is an open source project, distributed with the Apache software license. You can find the Nodyn source code on GitHub.

The Nodyn project is produced by the project:odd team, and a part of the JBoss polyglot ecosystem.

Related Projects

DynJS
The JS runtime that makes all of this work.
Netty
Netty provides the asynchronous glue.
